A cosa serve Bootstrap?

Bootstrap è un framework front-end che semplifica la creazione di pagine web responsive e moderne.

Cosa devi importare per creare una pagina web con Bootstrap?

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/5.3.0/css/bootstrap.min.css">
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server distribuiti che forniscono contenuti a utenti in diverse località. I vantaggi includono:

  • Velocità: caricamento più rapido grazie alla vicinanza del server.
  • Affidabilità: riduzione del rischio di downtime.
  • Caching: file già presenti nella cache del browser.

A cosa serve questo attributo del tag meta?

<meta name="viewport" content="width=device-width, initial-scale=1">

Questo attributo controlla il layout su dispositivi mobili, adattando la pagina alla larghezza del dispositivo.

Che differenza c’è tra Container e Container Fluid?

Container: larghezza fissa.

Container Fluid: 100% della larghezza disponibile.
